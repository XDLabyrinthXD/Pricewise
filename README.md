# 🛍️ PriceWise – Smart Price Comparison Platform

A full-stack web application that compares product prices across multiple e-commerce platforms in real-time, helping users find the best deals instantly.

---

## 🚀 Features

* 🔍 **Real-Time Price Comparison**
  Compare prices from multiple platforms like Amazon, Flipkart, JioMart, and more

* ⚡ **Fast Search System**
  Optimized backend scraping and parsing for quick results

* 🎯 **Smart Product Matching**
  Identifies the same product across different platforms

* 📊 **Clean & Interactive UI**
  Built with React for a smooth user experience

* 🧠 **Scalable Backend Architecture**
  Modular scraper design for adding new platforms easily

## ⚙️ How It Works

1. User searches for a product
2. Backend triggers multiple scrapers
3. Data is extracted and cleaned
4. Products are matched across platforms
5. Results are displayed with best price highlighted

## 🖥️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/XDLabyrinthXD/Pricewise.git
cd Pricewise
```

### 2️⃣ Backend Setup

```bash
cd BACKEND
pip install -r requirements.txt
python app.py
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
